## 📌 Overview

A personal professional portfolio website built to showcase skills, projects, and contact information. Designed with a **dark futuristic aesthetic** using pure HTML, CSS, and JavaScript — no frameworks required.

## ✨ Features

- **Responsive Design** — Works on desktop, tablet, and mobile
- **Custom Glowing Cursor** — Animated cursor with ring follower effect
- **Hero Section** — Animated name reveal with floating orb effects
- **About Section** — Personal bio with live stats cards
- **Skills Section** — Categorized tech stack (Frontend, Backend, Database, Tools)
- **Projects Section** — Cards for all 3 Future Interns tasks with GitHub links
- **Contact Form** — Functional form with success state animation
- **Scroll Reveal** — Elements animate in as you scroll
- **Active Nav Highlight** — Nav link highlights based on current section
- **SEO Friendly** — Meta tags and semantic HTML structure
